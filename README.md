# Gemini Live Voice Console (Frontend Only)

This is a simplified version of the Gemini Live Voice Console that runs entirely in the browser. It connects directly to the Google Gemini Multimodal Live API.

## Setup

1. Copy `.env.example` to `.env`.
2. Fill in your `VITE_GEMINI_API_KEY`.
3. Install dependencies and start the development server.

## Environment Variables

- `VITE_GEMINI_API_KEY`: Your Google Gemini API Key.

## Deployment

This app is ready to be deployed to Vercel. Make sure to add `VITE_GEMINI_API_KEY` to your Vercel project environment variables.
